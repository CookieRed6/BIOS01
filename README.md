# neko8

**Neko8** is a **fantasy console**, inspired by **pico8**, **liko12** and **tic80**.

Currently it features:

* Pico8-like api
* Code editor
* Sprite editor
* Terminal

More to come ;)

#### Installing ang running

Download and extract the source. In the root folder run:

```
love .
```

You will need **love2d** to run **neko8**.
